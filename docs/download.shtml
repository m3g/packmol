How to download and install Packmol

✓ Recommended: The main supported way to get Packmol running on any platform (Windows, MacOS, Linux) is to use the Julia wrapper for Packmol.

aInstall Julia

Download and install the Julia programming language from: https://julialang.org/downloads/

bInteractive Usage

b.1) Install Packmol:

Open Julia and type:

julia> import Pkg; Pkg.add("Packmol")

b.2) Running Packmol:

Inside the Julia REPL, type:

julia> using Packmol; run_packmol()

This will open a GUI from which you will be able to select the input file to be used by Packmol.

cCommand-line Interface

The ~/.julia/bin/packmol executable must be in your path after installation.

c.1) Install the Packmol app:

julia -e 'import Pkg; Pkg.Apps.add("Packmol")'

c.2) Run Packmol as standalone app from the shell:

packmol -i input.inp

Additional details can be found in the Packmol.jl page.
